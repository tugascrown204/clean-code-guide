Meaningful Naming

Choose descriptive names for variables and functions to improve readability.

const userAge = 25;

Functions Should Do One Thing

A function should have a single responsibility.

function fetchUserData() { /* fetch data */ }

Don't Repeat Yourself (DRY)

Abstract out repeated code to avoid duplication.

function calculateTax(amount) { return amount * 0.2; }

Commenting and Documentation

Use comments sparingly and only to explain why certain decisions were made.

// Calculate tax for the user

Code Formatting

Maintain consistent formatting and structure in your code.

if (isActive) { activate(); }
